Aegisub 快速概览

常见任务：

  • 用已翻译的文本制作音频时间轴
  • 对字幕进行简单的排版
  • 将完成的字幕附到视频上（用于观看或发布）
  • 将字幕译成另一种语言
  • 校正字幕使之与视频同步
  • 为歌曲制作卡拉OK时间轴
  • 创作华丽的卡拉OK特效字幕
