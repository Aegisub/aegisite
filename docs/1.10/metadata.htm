Meta data

    

The meta info contains general information about the subtitles. It roughly corresponds to the [Script Info] section in an ASS file. Currently, only two fields are defined:

 

res_x (number)

res_y (number)

The X and Y pixel resolution of the script. These define the "virtual screen" the subtitles are rendered on. There is currently no way to get the pixel aspect ratio of the actual video.
