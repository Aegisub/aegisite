About

 

Aegisub was developed using Microsoft Visual Studio 2003 and wxWidgets 2.6.0. The Automation module also uses the Lua Programming Language. The Regular Expressions code used in this program was written by Henry Spence and included in wxWidgets. Aegisub also uses the PortAudio, libpng, zlib and ffmpeg libraries.
