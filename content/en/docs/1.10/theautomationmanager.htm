The Automation Manager

    

The Automation Manager is used to load Automation scripts, assigning them to a subtitle file. It can also be used to create new scripts.

 

You access the Automation Manager through the Tools menu in the main window.

 

 

The buttons do the following:

 

Create...

Creates a new Automation script and loads it. This will start the editor you have selected. (See more about that below.)

 

Add...

Load an existing script.

 

Remove

Unloads the currently selected script.

 

Apply now

Run the selected script on the currently loaded subs. If the script supports configuration, you will be asked for settings first. You can undo the effect of running a script after closing the Automation Manager.

Note that using this button is generally discouraged; often it's more convenient to apply Automation scripts during an export operation. Automation scripts appear as export filters in the Export dialog.

 

Edit...

Starts the script editor. The first time you use this function, you will be asked to select an editor program used to edit scripts. You can select another editor by holding the Shift key while clicking this button.

 

Reload

Attempts to reload the currently selected script. This is required if you edit the script while Aegisub is open, in order for the changes to take effect.

 

Close

Closes the Automation Manager. Note that all changes are instant, you cannot cancel the changes made in this window.
